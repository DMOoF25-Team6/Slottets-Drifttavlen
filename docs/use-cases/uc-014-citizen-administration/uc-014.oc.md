## Operation Contract

### navigateToCitizenAdministration
- **Preconditions**: Administrator is authenticated and authorized; citizen administration logic is available.
- **Postconditions**: Citizen administration section is displayed to the administrator.

### selectCreateCitizen
- **Preconditions**: Citizen administration section is displayed.
- **Postconditions**: Citizen creation form is displayed to the administrator.

### submitCitizenForm
- **Preconditions**: Citizen creation form is displayed; administrator has entered citizen information.
- **Postconditions**: 
    - If information is missing or invalid: System highlights missing/invalid fields and prompts for correction.
    - If duplicate citizen: System displays an error and does not create the citizen.
    - If valid input: System creates a new citizen record, confirms creation, and displays the new citizen in the list. Audit trail records the creation event.
