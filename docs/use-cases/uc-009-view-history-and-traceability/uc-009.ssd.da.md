## Systemsekvensdiagram

```mermaid
sequenceDiagram
    actor AutoriseretMedarbejder
    participant System

    AutoriseretMedarbejder ->> System: åbnHistorikOgSporbarhed()

    alt brugerIkkeAutoriseret
        System -->> AutoriseretMedarbejder: visAutoriseringsfejl()

    else succes
        System -->> AutoriseretMedarbejder: visRegistreringerDerKanAuditeres(registreringstyperEllerSenesteRegistreringer)

        AutoriseretMedarbejder ->> System: vælgRegistrering(registreringId, tidsinterval)

        alt fejlVedIndlæsningAfHistorik
            System -->> AutoriseretMedarbejder: visFejlOgPrøvIgenMulighed()

        else ingenHistorikFundet
            System -->> AutoriseretMedarbejder: visIngenHistorikBesked()

        else succes
            System -->> AutoriseretMedarbejder: visHistorikliste(auditPoster)

            AutoriseretMedarbejder ->> System: vælgHistorikelement(auditPostId)

            alt fejlVedIndlæsningAfÆndringsdetaljer
                System -->> AutoriseretMedarbejder: visFejlOgReturnerTilHistorikliste()

            else succes
                System -->> AutoriseretMedarbejder: visÆndringsdetaljer(ændringsdetaljer, hændelsestid, registreringstid, erSenEllerTilbagevirkende)
            end
        end

        AutoriseretMedarbejder ->> System: lukHistorikOgSporbarhed()
        System -->> AutoriseretMedarbejder: returnerTilForrigeVisning()
    end
```
