## System Sequence Diagram

```mermaid
sequenceDiagram
    actor AuthorizedStaffMember
    participant System

    AuthorizedStaffMember ->> System: openHistoryAndTraceabilityView()

    alt userNotAuthorized
        System -->> AuthorizedStaffMember: displayAuthorizationError()

    else success
        System -->> AuthorizedStaffMember: displayAuditableRecords(recordTypesOrRecentRecords)

        AuthorizedStaffMember ->> System: selectRecord(recordId, timeRange)

        alt errorLoadingHistory
            System -->> AuthorizedStaffMember: displayErrorAndRetryOption()

        else noHistoryFound
            System -->> AuthorizedStaffMember: displayNoRecordsMessage()

        else success
            System -->> AuthorizedStaffMember: displayHistoryList(auditEntries)

            AuthorizedStaffMember ->> System: selectHistoryEntry(auditEntryId)

            alt errorLoadingChangeDetails
                System -->> AuthorizedStaffMember: displayErrorAndReturnToHistoryList()

            else success
                System -->> AuthorizedStaffMember: displayChangeDetails(changeDetails, eventTime, registeredTime, isLateOrRetroactive)
            end
        end

        AuthorizedStaffMember ->> System: closeHistoryAndTraceabilityView()
        System -->> AuthorizedStaffMember: returnToPreviousView()
    end
```
