## Wireframe

UC-009 giver en skrivebeskyttet visning til:
- At se tidligere overlap/vagtskifte ud fra dato og vagt.
- At se sporbarhed (audit trail) for ændringer: hvem ændrede hvad og hvornår.

Overordnet layout (adgang fra Drifttavlen):

```text
+----------------------------------------------------------------------------------+
|  SLOTTETS DRIFTTAVLE                                            [Vagt: D/A/N]   |
+----------------------------------------------------------------------------------+
|  [Hovedindhold på drifttavlen — beboerkort, fælles sektioner osv.]               |
|                                                                                  |
|  Handlinger:                                             [Historik & sporbarhed] |
+----------------------------------------------------------------------------------+
```

Detaljeret layout — Historik & Sporbarhed:

```text
+----------------------------------------------------------------------------------+
|  HISTORIK & SPORBARHED                                        [Tilbage]         |
+----------------------------------------------------------------------------------+
|  Vælg overlap (vagtskifte)                                                        |
|  Fra: [ÅÅÅÅ-MM-DD]  Til: [ÅÅÅÅ-MM-DD]  Vagt: [D/A/N/Alle]   [Anvend] [Nulstil]   |
|                                                                                  |
|  Overlaps (skrivebeskyttet liste)                                                 |
|  +----------------------------------------------------------------------------+  |
|  | Dato       | Vagt | Afdeling  | Overblik                    | [Åbn]      |  |
|  +----------------------------------------------------------------------------+  |
|  | 2026-05-07 | A    | Slottet   | 12 beboere, 4 opgaver, 3 noter | [Åbn]   |  |
|  | 2026-05-07 | D    | Slottet   | 12 beboere, 5 opgaver, 1 note  | [Åbn]   |  |
|  | 2026-05-06 | N    | Slottet   | 12 beboere, 3 opgaver, 2 noter | [Åbn]   |  |
|  +----------------------------------------------------------------------------+  |
|                                                                                  |
|  Valgt overlap (skrivebeskyttet snapshot)                                        |
|  +----------------------------------------------------------------------------+  |
|  | Overlap: 2026-05-07 / Aften / Slottet                                        |  |
|  |                                                                            |  |
|  |  [Skrivebeskyttet snapshot af overlap-layout (samme sektioner som tavlen)] |  |
|  |  - Beboere (initialer + status)                                             |  |
|  |  - Beskeder/noter                                                           |  |
|  |  - Opgaver                                                                  |  |
|  |  - Fælles sektioner (fx PhoneList)                                          |  |
|  +----------------------------------------------------------------------------+  |
|                                                                                  |
|  Sporbarhed (audit trail)                                                        |
|  Filtre: Entitet: [Alle ▼]  Ændring: [Alle ▼]  Søg: [              ]  [Filtrér]  |
|                                                                                  |
|  +----------------------------------------------------------------------------+  |
|  | Tidspunkt         | Entitet         | Ændring | Ændret af       | Beskrivelse |  |
|  +----------------------------------------------------------------------------+  |
|  | 2026-05-07 10:02  | ResidentNote    | Opdateret | user@slottet.dk | Redigeret  |  |
|  | 2026-05-07 09:15  | PhoneAssignment | Opdateret | user@slottet.dk | Tildelt tlf|  |
|  | 2026-05-06 22:01  | Task            | Oprettet  | user@slottet.dk | Tilføjet   |  |
|  +----------------------------------------------------------------------------+  |
|                                                                                  |
|  Valgt audit-hændelse (skrivebeskyttede detaljer)                                |
|  +----------------------------------------------------------------------------+  |
|  | Tidspunkt:   2026-05-07 10:02                                                |  |
|  | Entitet:     ResidentNote                                                    |  |
|  | Ændring:     Opdateret                                                       |  |
|  | Ændret af:   user@slottet.dk                                                 |  |
|  | Beskrivelse: [Redigeret note: "Skal have ekstra væske i dag"]                |  |
|  +----------------------------------------------------------------------------+  |
+----------------------------------------------------------------------------------+
```

- Al historik og audit-data er skrivebeskyttet i dette use case.
- “[Åbn]” viser det valgte overlap i området “Valgt overlap” (ingen separat modal/side).
- Audit-rækker viser: tidspunkt, hvad der ændrede sig (entitet), ændringstype, hvem der ændrede, og en kort beskrivelse.

## Noter
- Denne wireframe understøtter REQ-F-006 (historik + sporbarhed / audit trail).
- Overlap-listen vises som en separat skrivebeskyttet liste for at undgå at introducere redigeringshandlinger i UC-009.
- “Ændret af” vises som en bruger-identifikator (fx e-mail/brugernavn) for entydighed.
- Overbliks-tal er illustrative; formålet er at give et hurtigt overblik pr. vagt.
