## Sequence Diagram

### Presentation Layer → Application Layer
```mermaid
sequenceDiagram
    actor AuthorizedStaffMember as Actor
    participant WebUI as HistoryTraceabilityPage (WebUI)
    participant HistoryTraceabilityService as HistoryTraceabilityService (Core)

    AuthorizedStaffMember->>+WebUI: OpenHistoryAndTraceabilityView()

    WebUI->>+HistoryTraceabilityService: GetOverlapSummariesAsync(OverlapQueryDto)
    alt NotAuthorized
        HistoryTraceabilityService-->>WebUI: AuthorizationError()
        WebUI-->>AuthorizedStaffMember: DisplayAuthorizationError()
    else Success
        HistoryTraceabilityService-->>-WebUI: OverlapSummaryDto[]
        WebUI-->>AuthorizedStaffMember: DisplayOverlaps(OverlapSummaryDto[])
    end

    AuthorizedStaffMember->>+WebUI: OpenOverlap(overlapId)
    WebUI->>+HistoryTraceabilityService: GetOverlapSnapshotAsync(overlapId)
    HistoryTraceabilityService-->>-WebUI: OverlapSnapshotDto
    WebUI-->>AuthorizedStaffMember: DisplayOverlapSnapshot(OverlapSnapshotDto)

    AuthorizedStaffMember->>+WebUI: SelectRecordForTraceability(recordId, timeRange, filters)
    WebUI->>+HistoryTraceabilityService: GetAuditEntriesAsync(AuditQueryDto)

    alt ErrorLoadingHistory
        HistoryTraceabilityService-->>WebUI: Error(message)
        WebUI-->>AuthorizedStaffMember: DisplayErrorAndRetryOption()
    else NoHistoryFound
        HistoryTraceabilityService-->>WebUI: EmptyResult()
        WebUI-->>AuthorizedStaffMember: DisplayNoRecordsMessage()
    else Success
        HistoryTraceabilityService-->>-WebUI: AuditEntrySummaryDto[]
        WebUI-->>AuthorizedStaffMember: DisplayHistoryList(AuditEntrySummaryDto[])
    end

    AuthorizedStaffMember->>+WebUI: SelectAuditEntry(auditEntryId)
    WebUI->>+HistoryTraceabilityService: GetAuditEntryDetailsAsync(auditEntryId)

    alt ErrorLoadingChangeDetails
        HistoryTraceabilityService-->>WebUI: Error(message)
        WebUI-->>AuthorizedStaffMember: DisplayErrorAndReturnToHistoryList()
    else Success
        HistoryTraceabilityService-->>-WebUI: AuditEntryDetailsDto(eventTime, registeredTime, isLateOrRetroactive, changeDetails)
        WebUI-->>AuthorizedStaffMember: DisplayChangeDetails(AuditEntryDetailsDto)
    end

    AuthorizedStaffMember->>+WebUI: CloseHistoryAndTraceabilityView()
    WebUI-->>-AuthorizedStaffMember: ReturnToPreviousView()
```

### Application Layer → Infrastructure Layer (External Interfaces)
```mermaid
sequenceDiagram
    participant HistoryTraceabilityService as HistoryTraceabilityService (Core)
    participant HistoryTraceabilityManager as HistoryTraceabilityManager (Infrastructure)

    HistoryTraceabilityService->>+HistoryTraceabilityManager: GetOverlapSummariesAsync(OverlapQueryDto)
    HistoryTraceabilityManager-->>-HistoryTraceabilityService: OverlapSummaryDto[]

    HistoryTraceabilityService->>+HistoryTraceabilityManager: GetOverlapSnapshotAsync(overlapId)
    HistoryTraceabilityManager-->>-HistoryTraceabilityService: OverlapSnapshotDto

    HistoryTraceabilityService->>+HistoryTraceabilityManager: GetAuditEntriesAsync(AuditQueryDto)
    alt Success
        HistoryTraceabilityManager-->>HistoryTraceabilityService: AuditEntrySummaryDto[]
    else NoHistoryFound
        HistoryTraceabilityManager-->>HistoryTraceabilityService: []
    else NotAuthorized
        HistoryTraceabilityManager-->>HistoryTraceabilityService: AuthorizationError()
    else Error
        HistoryTraceabilityManager-->>-HistoryTraceabilityService: Error(message)
    end

    HistoryTraceabilityService->>+HistoryTraceabilityManager: GetAuditEntryDetailsAsync(auditEntryId)
    HistoryTraceabilityManager-->>-HistoryTraceabilityService: AuditEntryDetailsDto
```

### WebApi Layer → Infrastructure Layer (Data Access)
```mermaid
sequenceDiagram
    participant HistoryTraceabilityManager as HistoryTraceabilityManager (Infrastructure)
    participant WebApi as HistoryTraceabilityController (Api)
    participant OverlapRepository as OverlapRepository (Infrastructure.Data)
    participant AuditEntryRepository as AuditEntryRepository (Infrastructure.Data)

    HistoryTraceabilityManager->>+WebApi: GET /api/history/overlaps (OverlapQueryDto)
    WebApi->>+OverlapRepository: GetOverlapSummariesAsync(from, to, shift)
    OverlapRepository-->>-WebApi: OverlapSummaryDto[]
    WebApi-->>-HistoryTraceabilityManager: 200 OK (OverlapSummaryDto[])

    HistoryTraceabilityManager->>+WebApi: GET /api/history/overlaps/{overlapId}/snapshot
    WebApi->>+OverlapRepository: GetOverlapSnapshotAsync(overlapId)
    OverlapRepository-->>-WebApi: OverlapSnapshotDto
    WebApi-->>-HistoryTraceabilityManager: 200 OK (OverlapSnapshotDto)

    HistoryTraceabilityManager->>+WebApi: GET /api/history/audit (AuditQueryDto)
    alt NotAuthorized
        WebApi-->>HistoryTraceabilityManager: 401/403
    else Success
        WebApi->>+AuditEntryRepository: GetAuditEntriesAsync(query)
        AuditEntryRepository-->>-WebApi: AuditEntrySummaryDto[]
        WebApi-->>HistoryTraceabilityManager: 200 OK (AuditEntrySummaryDto[])
    else NoHistoryFound
        WebApi->>+AuditEntryRepository: GetAuditEntriesAsync(query)
        AuditEntryRepository-->>-WebApi: []
        WebApi-->>HistoryTraceabilityManager: 200 OK ([])
    else Error
        WebApi-->>HistoryTraceabilityManager: 5xx (Error)
    end

    HistoryTraceabilityManager->>+WebApi: GET /api/history/audit/{auditEntryId}
    alt NotFound
        WebApi-->>HistoryTraceabilityManager: 404 Not Found
    else Success
        WebApi->>+AuditEntryRepository: GetAuditEntryDetailsAsync(auditEntryId)
        AuditEntryRepository-->>-WebApi: AuditEntryDetailsDto
        WebApi-->>HistoryTraceabilityManager: 200 OK (AuditEntryDetailsDto)
    end
```

## Notes
- UC-009 is read-only: viewing history does not modify business data.
- Late/retroactive entries are presented using both `eventTime` and `registeredTime` (REQ-F-007).
- DTOs are used across boundaries to avoid leaking domain entities:
  - `OverlapQueryDto`, `OverlapSummaryDto`, `OverlapSnapshotDto`
  - `AuditQueryDto`, `AuditEntrySummaryDto`, `AuditEntryDetailsDto`
- Clean Architecture dependency direction is preserved (Infrastructure → Application → Domain).
