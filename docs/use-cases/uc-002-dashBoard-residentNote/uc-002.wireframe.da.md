## Wireframe

Drifttavlen viser beboerkort. Når en medarbejder vælger en beboer, viser systemet aktuelle beskeder/noter for den beboer og giver mulighed for at tilføje, redigere og slette beskeder/noter.

Overordnet layout (Drifttavle med valgt beboer):

```text
+----------------------------------------------------------------------------------+
|  SLOTTETS DRIFTTAVLE                                            [Vagt: D/A/N]   |
+----------------------------------------------------------------------------------+
|  Beboere (kort)                                                                  |
|  +----------------+  +----------------+  +----------------+  +----------------+  |
|  | Initialer: AB  |  | Initialer: CD  |  | Initialer: EF  |  | Initialer: GH  |  |
|  | Beskeder: (2)  |  | Beskeder: (0)  |  | Beskeder: (1)  |  | Beskeder: (0)  |  |
|  +----------------+  +----------------+  +----------------+  +----------------+  |
|                                                                                  |
|  Valgt beboer (denne wireframe)                                                  |
|  +--------------------------------------------------------------------------+    |
|  | Beboer: AB (kun initialer)                                              |    |
|  | Beskeder/Noter                                                          |    |
|  | [område til bekræftelse / fejlmeddelelse]                               |    |
|  +--------------------------------------------------------------------------+    |
+----------------------------------------------------------------------------------+
```

Detaljeret layout — Beskeder/Noter for valgt beboer:

```text
+----------------------------------------------------------------------------------+
|  Beboer: AB (Initialer)                                           [Beskeder]    |
+----------------------------------------------------------------------------------+
|  Aktuelle beskeder/noter                                                         |
|  +--------------------------------------------------------------------------+    |
|  | Besked/Note-tekst                                               Handling |    |
|  +--------------------------------------------------------------------------+    |
|  | "Skal have ekstra væske i dag" (Redigeret: 09:15 20-04-2026)      [Redigér] [Slet] |
|  | "Familiebesøg kl. 15:00" (Redigeret: 10:02 20-04-2026)            [Redigér] [Slet] |
|  +--------------------------------------------------------------------------+    |
|                                                                                  |
|  Tilføj besked/note                                                              |
|  Besked/Note: [                                                              ]  |
|              [                                                              ]  |
|              [                                                              ]  |
|              [Tilføj]                                                          |
|                                                                                  |
|  Feedback                                                                        |
|  Bekræftelse: "Besked gemt" (vises efter tilføj/redigér/slet)                   |
|  Fejl:        "Besked kunne ikke gemmes" (vises ved gemmefejl)                  |
+----------------------------------------------------------------------------------+
```

Redigér-tilstand (inline for én række):

```text
+--------------------------------------------------------------------------+
| "[                                                           ]" [Gem] [Annullér] |
+--------------------------------------------------------------------------+
```

- Initialer er skrivebeskyttede (ingen fulde navne).
- Beskeder/noter vises pr. beboer; listen kan være tom eller indeholde flere rækker.
- Tilføj/Redigér/Slet er de eneste handlinger i dette use case.

## Noter
- Denne wireframe afspejler UC-002: se aktuelle beskeder, tilføje, redigere, slette, samt vise bekræftelse/fejl.
- Hver besked/note viser EditedAt-tidsstemplet fra ResidentNoteDto.
- Der vises ikke ekstra attributter (fx forfatter), da de ikke er nævnt som krav i UC-002.
- Placeringen vises som en sektion på drifttavlen for at undgå at introducere nye skærme/modaler, som ikke er beskrevet i UC-002.
