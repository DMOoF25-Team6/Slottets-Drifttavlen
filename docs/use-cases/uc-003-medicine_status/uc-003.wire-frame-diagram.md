## Notes
- This wireframe represents the user interface for viewing medicine and painkiller status of a resident.  
- The employee selects a resident and is presented with an overview of medicine administration and painkiller status.
- This wireframe reflects UC-003: view medicine and painkiller status for a selected resident.
- The medicine data corresponds to records from the last 24 hours.
- The painkiller section shows timing constraints for safe administration.
- No create, update, or delete actions are included, as this use case is read-only.
- The layout is presented within the dashboard to remain consistent with existing system design.
