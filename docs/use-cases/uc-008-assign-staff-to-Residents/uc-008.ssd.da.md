## Systemsekvensdiagram
```mermaid
sequenceDiagram
    actor AutoriseretPersonalemedlem
    participant System

    AutoriseretPersonalemedlem ->> System: åbnPersonaletildelingsoversigt(vagt)

    alt brugerIkkeAutoriseret
        System -->> AutoriseretPersonalemedlem: visAutoriseringsfejl()

    else fejlVedIndlæsningAfTildelingsdata
        System -->> AutoriseretPersonalemedlem: visFejlbesked()

    else succes
        System -->> AutoriseretPersonalemedlem: visBeboereOgTilgængeligtPersonale(beboere, personale)
    end

    AutoriseretPersonalemedlem ->> System: tildelPersonaleTilBeboer(beboer, personalemedlem, vagt)

    alt beboerHarIngenTildeltPersonale
        System -->> AutoriseretPersonalemedlem: visValideringsadvarsel()

    else fejlVedGemningAfTildeling
        System -->> AutoriseretPersonalemedlem: visFejlOgPrøvIgenMulighed()

    else succes
        System -->> AutoriseretPersonalemedlem: visTildelingsoversigt()
    end
```

## Noter
- Systemet validerer, at beboeren har mindst ét tildelt personalemedlem.
- Kun autoriserede roller kan oprette eller opdatere personaletildelinger.
- Ændringer i personaletildelinger logges i audit trail.
- Den opdaterede tildelingsoversigt viser ansvar under vagten.
